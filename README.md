# Web3 Blockchain Tools Suite
一套功能完整的区块链Web工具集合，支持钱包管理、合约校验、NFT查看、跨链桥、质押计算、投票、Gas追踪等多场景Web3功能，纯前端轻量化部署，开箱即用。

## 项目特性
- 纯HTML+JavaScript开发，无需后端依赖
- 支持多链（ETH/BSC/Polygon/Avalanche）适配
- 完整Web界面，操作简单直观
- 工具类型覆盖钱包、交易、NFT、DAO、质押、空投、桥接、存储等全场景
- 轻量、高效、可直接部署静态页面

## 文件列表与功能说明
1. BlockchainWalletGenerator.html - 区块链钱包生成器，支持地址、私钥、助记词一键生成
2. TransactionSignerTool.js - 交易签名工具，支持交易构造、签名、验签功能
3. NFTMetadataViewer.html - NFT元数据查看器，输入合约与TokenID查询NFT信息
4. ChainGasTracker.js - 多链Gas价格追踪器，实时获取慢/标准/快速Gas费
5. DecentralizedStorageUploader.html - 去中心化存储上传工具，支持IPFS上传与链接生成
6. SmartContractChecker.js - 智能合约校验工具，支持地址验证、类型识别、风险扫描
7. BlockchainBlockExplorer.html - 区块链区块浏览器，查询区块高度、哈希、交易数量
8. TokenBalanceChecker.js - 代币余额查询器，支持原生资产与Token批量查询
9. DAOProposalVoting.html - DAO提案投票系统，支持链上投票与结果实时展示
10. CrossChainBridgeTool.js - 跨链桥工具，支持跨链转账预估、订单创建
11. StakingRewardCalculator.html - 质押收益计算器，支持自定义数量与周期计算收益
12. AirdropClaimTool.js - 空投领取工具，支持资格检查与空投发放
13. WhitelistManagerDApp.html - 白名单管理DApp，支持添加与查询白名单资格
14. MarketOrderBookSimulator.js - 市场订单簿模拟器，生成买卖盘口与实时价格
15. Web3WalletConnector.html - Web3钱包连接器，支持前端连接钱包、展示地址与余额

## 使用方式
直接打开任意HTML文件即可在浏览器运行，JS工具可通过导入使用，所有工具均为本地运行，无数据上传风险。

## 适用场景
- Web3前端学习与演示
- 区块链工具快速开发
- 个人钱包/交易/NFT管理
- DAO、质押、空投等DApp原型展示
